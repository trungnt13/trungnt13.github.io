# trungnt13.github.io

Personal blog featuring minimalist design and interactive JavaScript demonstrations.

## Design Principles

1. **Minimalist and Clutter-Free** - Clean layout with focus on content
2. **Focused and Intuitive** - Simple navigation and clear hierarchy
3. **Maximized Reader Attention** - Typography and spacing optimized for readability
4. **Interactive Demonstrations** - Support for JavaScript canvas visualizations

## Structure

- `index.html` - Homepage with bio and interests
- `blog/index.html` - Blog post listing
- `blog/*.html` - Individual blog posts
- `styles.css` - Minimalist CSS styling

## Features

- Responsive design (mobile and desktop)
- Clean typography with system fonts
- Interactive HTML5 Canvas demos
- Semantic HTML structure
- Fast loading with no external dependencies

## Adding New Blog Posts

1. Create a new HTML file in the `blog/` directory
2. Use the existing blog post templates as a reference
3. Add the post to `blog/index.html` listing
4. Follow the existing design patterns for consistency

## Local Development

Serve the files with any static web server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.
