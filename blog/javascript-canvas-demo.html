JavaScript Canvas Demo

This post demonstrates the interactive capabilities of this blog platform. Below you'll find a simple particle system simulation built with HTML5 Canvas and JavaScript.

Interactive Particle System

The canvas below shows an animated particle system. Particles move around the canvas, creating organic patterns. This demonstrates how we can integrate interactive visualizations into blog posts.

How It Works

The simulation uses the following concepts:

  • Particle Objects: Each particle has position, velocity, and size properties
  • Animation Loop: Using requestAnimationFrame for smooth 60fps rendering
  • Physics: Simple velocity-based movement with boundary detection
  • Canvas API: Drawing circles and clearing the canvas each frame

Code Structure

The implementation consists of:

class Particle {
    constructor(x, y) {
        this.x = x;
        this.y = y;
        this.vx = (Math.random() - 0.5) * 2;
        this.vy = (Math.random() - 0.5) * 2;
        this.radius = Math.random() * 3 + 1;
    }
    
    update(width, height) {
        this.x += this.vx;
        this.y += this.vy;
        
        // Bounce off edges
        if (this.x < 0 || this.x > width) this.vx *= -1;
        if (this.y < 0 || this.y > height) this.vy *= -1;
    }
}
